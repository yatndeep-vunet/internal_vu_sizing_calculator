How to Use VuNet Sizing Calculator

1

Create Template

Design scenario-specific templates for different infrastructure sizing needs. Each template can be customized to represent unique deployment environments.

Example:

Cloud Deployment, Enterprise Setup, SMB Infrastructure

2

Enter Input

Input key parameters like user count, data volume, network traffic, and application requirements into your selected template.

Key Inputs:
  • Total Users
  • Monthly Data Volume
  • Peak Concurrent Users
3

Export Results

Generate comprehensive reports and export detailed sizing recommendations directly to Excel for further analysis and sharing.

Export Formats:

.xlsx, .csv with complete infrastructure recommendations

4

Save Template

Store your input configurations as reusable templates for quick future sizing calculations across similar infrastructure scenarios.

Template Management:

Name, categorize, and quickly retrieve saved configurations
